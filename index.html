#ProjectARMADAV

Radiation cannot be detected easily and manned testing requires resources and manpower. However, by using a robot to do the job requires less resources making it more efficient than manned testing. Learn More

Can Upload to Safecast

The bGeigie Nano onboard the ARMADAV logs data into an SD card which can be uploaded to the Safecast Database which can be seen on a map

Award-Winning

The ARMADAV was nationally recognized as 2nd place in the 2019 National Robotics Competition in the Philippines, and internationally declared Champion (Platinum Award) in the 2019 Young Inventors Challenge in Cyberjaya, Malaysia. Learn More
